# mutex

## 使用方法
1. 包含头文件  
```cpp
#include <mutex>
```

2. 在类中定义一个std::mutex变量  
```cpp
class Example {
private:
    std::mutex data_mutex_;
};
```

3. 在类中函数中定义一个局部变量，在函数的作用域结束后，自动会解锁（通过析构函数）
```cpp
std:lock_guard<std::muxtex> lock(data_mutex_)
```

## 作用与优势
- 自动管理锁，减少手动加锁解锁的错误风险。  
- 确保线程安全，防止数据竞争和竞态条件。  
- 适合不需要动态解锁的场景，代码简洁高效。
