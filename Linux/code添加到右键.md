# Code 添加到右键
<b><font size=4 > <!-- Markdown不支持精细的字体设置 -->

1. ctrl+Alt+T 打开一个终端
1. ` cd ~/.local/share/nautilus/scripts `
2. ` vim "Open in vscode" `
3. 在“Open in vscode”中输入 <br>
  ` code ./$1 `
4. sudo chmod u+x "Open in vscode"
5. 在要打开的文件夹上右键 在scripts上选择 Open in vscode
