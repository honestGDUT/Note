# 命令行指令
```
df -h
```
- df命令直接执行的效果不好，不能以mb或者gb的形式显示磁盘容量。为了以人性化显示，可以在后面加-h参数，h是human-reading的第一个字母，可以理解为人类能读的格式。

- 要查看所有文件系统的磁盘空间，可以使用-a参数，同时-h和-a参数可以合用。
