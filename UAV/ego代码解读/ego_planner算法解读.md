其中 `Q` 是控制点集合，`λ` 是权重系数。

#### 平滑度代价 J_s
- 使用**加加速度**的积分作为平滑度代价
- 数学表达：`J_s = Σ ‖ d³p(t_i)/dt³ ‖²`
- 特点：可写成控制点的二次型形式，便于快速求解

#### 碰撞代价 J_c - 创新点1
**传统方法问题**：
- 使用距离场梯度，计算量大
- 容易陷入局部最小值

**EGO-Planner解决方案**：
1. **"推离"思想**：为离障碍物太近的控制点找到"斥力点"
2. **惩罚函数**：`J_c = Σ F(d(c_i))`
3. **效果**：产生将控制点从障碍物表面"推开"的梯度

#### 动力学可行性代价 J_d
- 约束轨迹的速度、加速度等参数
- 数学表达：`J_d = Σ w_v F(‖v_i‖ - v_max) + Σ w_a F(‖a_i‖ - a_max)`
- 将硬约束转化为软约束，简化优化问题

### 3. 优化求解：数值优化方法

- **算法**：使用 **L-BFGS** 算法
- **问题类型**：非线性最小二乘问题
- **优势**：
  - 近似Hessian矩阵的逆，加速收敛
  - 内存消耗小，适合变量多的问题
  - 通过梯度下降调整控制点位置

### 4. 局部重规划与适应性 - 创新点2

处理未知障碍物的流程：
1. **轨迹截断**：截取尚未执行的安全部分
2. **设置新目标点**：在截断轨迹末端设置临时目标点
3. **重新规划**：调用前端和后端生成新局部轨迹
4. **轨迹融合**：平滑融合新旧轨迹，保证连续性

## 三、算法总结与优势

| 组件 | 使用技术 | 优势 |
|------|----------|------|
| **轨迹表示** | 均匀B样条 | 凸包性保障安全，高阶连续，局部支撑利于重规划 |
| **前端** | A*/JPS | 快速找到无碰撞初始路径，提供良好初值 |
| **后端优化** | 梯度下降(L-BFGS) | 高效求解非线性问题，实时性强 |
| **碰撞处理** | 基于弹性带的惩罚函数 | 避免局部最小，主动推离障碍物，比ESDF更高效 |
| **动力学约束** | 对控制点的惩罚项 | 将硬约束转化为软约束，简化优化问题 |
| **局部重规划** | 轨迹截断与优化 | 对未知环境反应迅速，鲁棒性强 |

## 核心贡献

1. **弹性带思想**：通过自适应惩罚函数解决梯度优化中的局部最小值和初值敏感问题
2. **高效优化**：整个优化问题构建高效，能在机载计算资源上实时运行
3. **轻量级**：不依赖高精度ESDF地图，降低计算和存储开销
4. **实用性**：适合资源受限的嵌入式平台，在真实场景中表现优异

EGO-Planner成功地将复杂运动规划问题转化为可快速求解的数值优化问题，在保证安全性和动态可行性的同时，实现了卓越的实时性能。
